利用js实现背景帧动画

发表于 2017-04-20 | 分类于 Coding

项目中需要用到帧动画的效果，第一反应是用css3的animation去实现。发现写出来的效果不理想，于是决定还是用js写一下。
思路是每次改变一定量的背景图片position的值

function bgFrame(el,x){
    for (var i = 0; i < 5; i++) {
        (function(i) {
            setTimeout(function() {
                i = (i-4)*x;
                $(el).css("background-position","0 "+i+"px");
            }, i*100);
        })(i);
    }
}
